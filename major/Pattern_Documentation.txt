PATTERN OVERVIEW:
================
The indicator creates horizontal lines based on three types of 45-degree patterns:

1. LONG-TERM PATTERNS (RED BOLD LINES)
2. SHORT-TERM PATTERNS (YELLOW LINES) 
3. REGULAR 45° MULTIPLES (YELLOW LINES)

===============================================================================
1. LONG-TERM PATTERNS (RED BOLD LINES)
===============================================================================

A) PRIORITY PATTERNS (Key Gann Levels):
---------------------------------------
45, 90, 135, 180, 225, 270, 315, 360, 405, 450, 495, 540, 585, 630, 675, 720, 765, 810, 855, 900...

These are the fundamental 45-degree astronomical/astrological levels.

B) xxx00 PATTERNS (5-digit numbers):
------------------------------------
Pattern: First 3 digits divisible by 45, last 2 digits are 00

Examples:
- 04500 (045 ÷ 45 = 1, ends in 00) ✓
- 09000 (090 ÷ 45 = 2, ends in 00) ✓
- 13500 (135 ÷ 45 = 3, ends in 00) ✓
- 18000 (180 ÷ 45 = 4, ends in 00) ✓
- 22500 (225 ÷ 45 = 5, ends in 00) ✓
- 27000 (270 ÷ 45 = 6, ends in 00) ✓
- 31500 (315 ÷ 45 = 7, ends in 00) ✓
- 36000 (360 ÷ 45 = 8, ends in 00) ✓
- 40500 (405 ÷ 45 = 9, ends in 00) ✓
- 45000 (450 ÷ 45 = 10, ends in 00) ✓

C) xx00 PATTERNS (4-digit numbers):
-----------------------------------
Pattern: First 2 digits divisible by 45, last 2 digits are 00

Examples:
- 4500 (45 ÷ 45 = 1, ends in 00) ✓
- 9000 (90 ÷ 45 = 2, ends in 00) ✓
- 1350 (Not valid - 13 ÷ 45 ≠ integer)
- 1800 (Not valid - 18 ÷ 45 ≠ integer)

Note: Only works when first 2 digits are exactly divisible by 45

D) xxx PATTERNS (3-digit numbers):
----------------------------------
Pattern: Exact multiples of 45

Examples:
- 045, 090, 135, 180, 225, 270, 315, 360, 405, 450, 495, 540, 585, 630, 675, 720, 765, 810, 855, 900...

E) xxx000 PATTERNS (6-digit numbers):
-------------------------------------
Pattern: First 3 digits divisible by 45, last 3 digits are 000

Examples:
- 045000, 090000, 135000, 180000, 225000, 270000, 315000, 360000...

===============================================================================
2. SHORT-TERM PATTERNS (YELLOW LINES)
===============================================================================

A) yyxxx PATTERNS (5-digit numbers):
------------------------------------
Pattern: Last 3 digits divisible by 45, first 2 can be anything

Examples:
- 12045 (last 3: 045 ÷ 45 = 1) ✓
- 34090 (last 3: 090 ÷ 45 = 2) ✓
- 56135 (last 3: 135 ÷ 45 = 3) ✓
- 78180 (last 3: 180 ÷ 45 = 4) ✓
- 91225 (last 3: 225 ÷ 45 = 5) ✓
- 23270 (last 3: 270 ÷ 45 = 6) ✓
- 45315 (last 3: 315 ÷ 45 = 7) ✓
- 67360 (last 3: 360 ÷ 45 = 8) ✓
- 89405 (last 3: 405 ÷ 45 = 9) ✓
- 11450 (last 3: 450 ÷ 45 = 10) ✓

B) yxxx PATTERNS (4-digit numbers):
-----------------------------------
Pattern: Last 3 digits divisible by 45, first digit can be anything

Examples:
- 1045 (last 3: 045 ÷ 45 = 1) ✓
- 2090 (last 3: 090 ÷ 45 = 2) ✓
- 3135 (last 3: 135 ÷ 45 = 3) ✓
- 4180 (last 3: 180 ÷ 45 = 4) ✓
- 5225 (last 3: 225 ÷ 45 = 5) ✓
- 6270 (last 3: 270 ÷ 45 = 6) ✓
- 7315 (last 3: 315 ÷ 45 = 7) ✓
- 8360 (last 3: 360 ÷ 45 = 8) ✓
- 9405 (last 3: 405 ÷ 45 = 9) ✓

C) yyyxxx PATTERNS (6-digit numbers):
-------------------------------------
Pattern: Last 3 digits divisible by 45, first 3 can be anything

Examples:
- 123045, 456090, 789135, 321180, 654225, 987270...

===============================================================================
3. REGULAR 45° MULTIPLES (YELLOW LINES)
===============================================================================

Pattern: Numbers divisible by 45 that don't match any special patterns

Examples:
- 135 (45 × 3, but also priority pattern - becomes LONG-TERM)
- 315 (45 × 7, regular multiple) ✓
- 405 (45 × 9, regular multiple) ✓
- 495 (45 × 11, regular multiple) ✓
- 585 (45 × 13, regular multiple) ✓
- 675 (45 × 15, regular multiple) ✓
- 765 (45 × 17, regular multiple) ✓
- 855 (45 × 19, regular multiple) ✓
- 945 (45 × 21, regular multiple) ✓
- 1035 (45 × 23, regular multiple) ✓
- 1125 (45 × 25, regular multiple) ✓
- 1215 (45 × 27, regular multiple) ✓

===============================================================================
PATTERN HIERARCHY & CONFLICT RESOLUTION
===============================================================================

1. PRIORITY PATTERNS always take precedence (LONG-TERM RED)
2. xxx00 patterns take precedence over yyxxx (LONG-TERM RED)
3. yyxxx patterns are SHORT-TERM YELLOW
4. Regular multiples are REGULAR YELLOW

No conflicts exist between xxx00 and yyxxx since xxx00 requires ending in 00.

===============================================================================
VISUAL REPRESENTATION
===============================================================================

RED BOLD LINES (Width 3):     LONG-TERM patterns (xxx00, priority levels)
YELLOW LINES (Width 1):       SHORT-TERM patterns (yyxxx) + REGULAR multiples

===============================================================================
ADDITIONAL GANN LEVELS (Between Main Lines)
===============================================================================

Between consecutive 45° lines, the indicator also draws:
- Gann Eighths (12.5%, 87.5%) - Brown lines
- Blue Eighths (37.5%, 62.5%) - Cyan lines  
- Gann Quarters (25%, 75%) - Orange lines
- Gann Half (50%) - Purple lines

Between consecutive RED BOLD lines:
- Bold Eighths (37.5%, 62.5%) - Magenta lines
- Bold Quarters (25%, 75%) - Teal lines
- Bold Special Eighths (12.5%, 87.5%) - Maroon lines
- Bold Half (50%) - Violet lines

===============================================================================
CONTROL TOGGLES
===============================================================================

- ShowBoldPatterns: Show/hide LONG-TERM red bold lines
- ShowRegularLines: Show/hide SHORT-TERM yellow lines
- ShowLongTermLines: Show/hide all long-term lines
- ShowShortTermLines: Show/hide all short-term lines
- ShowGannEighths: Show/hide Gann eighths between yellow lines
- ShowBlueEighths: Show/hide blue eighths between yellow lines
- ShowBoldEighths: Show/hide bold eighths between red lines
- ShowGannQuarters: Show/hide quarters between yellow lines
- ShowGannHalf: Show/hide half levels between yellow lines
